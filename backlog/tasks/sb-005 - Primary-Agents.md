## Description

Implement the 6 primary agents as defined in AGENT_ARCHITECTURE.md:
1. **Orchestrator** - Central coordinator, task routing
2. **Coder** - Code generation, file editing
3. **Reviewer** - Code review, best practices
4. **Planner** - Task breakdown, architecture
5. **Debugger** - Error diagnosis, fix suggestions
6. **Explorer** - Codebase navigation, search

## Acceptance Criteria

- [ ] All 6 primary agents implemented
- [ ] Each agent has specialized system prompt
- [ ] Each agent has appropriate tool set
- [ ] Inter-agent delegation working
- [ ] Agent-specific skills attached
