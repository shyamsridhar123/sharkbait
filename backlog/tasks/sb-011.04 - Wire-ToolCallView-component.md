## Description

The `ToolCallView` component exists in `src/ui/tool-call.tsx` but is NOT used in `app.tsx`. Currently tool execution just shows plain text like `⚡ read_file...`.

**Current behavior:**
```tsx
case "tool_start":
  toolOutput += `⚡ ${toolInfo}...\n`;  // Plain text
```

**Expected behavior:**
- Use the `ToolCallView` component with proper status (running/success/error)
- Show duration after completion
- Show truncated result on success
- Show error message on failure

## Acceptance Criteria

- [ ] `app.tsx` imports and uses `ToolCallView` component
- [ ] Running tools show spinner animation
- [ ] Completed tools show ✓ with duration
- [ ] Failed tools show ✗ with error message
- [ ] Tool results are truncated to 80 chars

## Technical Notes

The component already exists at `src/ui/tool-call.tsx` with all necessary props:
- `name: string`
- `status: "running" | "success" | "error"`
- `result?: string`
- `error?: string`
- `duration?: number`

Need to track tool start times to calculate duration.
