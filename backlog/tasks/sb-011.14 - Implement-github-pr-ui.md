## Technical Notes

1. GitHub tool result formatting:
```tsx
function formatGitHubResult(result: any): React.JSX.Element {
  if (result.number && result.url) {
    return (
      <Text>
        ✓ Created PR #{result.number}: "{result.title}"
        <Text color="blue"> {result.url}</Text>
      </Text>
    );
  }
}
```

2. Clickable links in terminals:
- Most modern terminals support OSC 8 hyperlinks
- ink-link package for React-based linking
- Fallback: show URL as plain text

3. Confirmation before PR:
```
Agent: Here are the changes I made:
       [Shows summary]
       Shall I create a PR? [y/n]
```
