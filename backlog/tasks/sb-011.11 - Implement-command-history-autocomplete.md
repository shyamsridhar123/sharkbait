## Description

PRD Section 5.6 requires: "Command history and autocomplete"

Currently:
- No history (up/down arrows don't work)
- No autocomplete (tab does nothing)

## Acceptance Criteria

- [ ] Up/Down arrows navigate command history
- [ ] Tab autocompletes slash commands
- [ ] Tab autocompletes file paths
- [ ] History persists across sessions (save to file)
- [ ] Fuzzy matching for autocomplete

## Technical Notes

### History
1. Store in `~/.sharkbait/history.txt`
2. Limit to last 1000 commands
3. Navigate with up/down arrows
4. Search with Ctrl+R (optional)

### Autocomplete
1. Detect context:
   - `/` at start → slash command completion
   - After `/cd ` → path completion
   - After file tool args → path completion
2. Use `ink-tab-complete` or custom implementation
3. Show suggestion dropdown
