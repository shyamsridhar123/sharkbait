## Description

Implement 12 slash commands for quick actions:

1. **/plan** - Create implementation plan
2. **/implement** - Start feature implementation
3. **/review** - Review code or PR
4. **/debug** - Diagnose an error
5. **/test** - Generate or run tests
6. **/refactor** - Start refactoring workflow
7. **/commit** - Smart commit with message
8. **/pr** - Create pull request
9. **/explain** - Explain code section
10. **/search** - Search codebase
11. **/status** - Show current task status
12. **/help** - Show available commands
