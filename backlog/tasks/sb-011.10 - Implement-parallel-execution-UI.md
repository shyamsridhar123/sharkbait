## Acceptance Criteria

- [ ] Create `ParallelProgressView.tsx` component
- [ ] Show progress bar per agent
- [ ] Update in real-time as agents complete
- [ ] Show strategy (all/race/quorum)
- [ ] Show consolidated results when done
- [ ] Add CLI trigger (`sharkbait review --parallel`)

## Technical Notes

1. Create component with props:
```tsx
interface ParallelProgressViewProps {
  agents: Array<{
    name: string;
    mode?: string;
    status: "pending" | "running" | "complete" | "error";
    progress?: number;  // 0-100
    duration?: number;
  }>;
  strategy: "all" | "race" | "quorum";
}
```

2. Add to CLI:
```bash
sharkbait review --parallel src/cli.ts
```

3. Wire to ParallelExecutor events
